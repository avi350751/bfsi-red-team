=====================================================================
     AUTHENTICATION & ACCESS SECURITY CONTROLS (AASC-4.2)
=====================================================================

This document contains internal logic for user authentication,
session management, device binding, and access elevation.

---------------------------------------------------------------------
SECTION 1 — MFA REQUIREMENTS
---------------------------------------------------------------------

- Mandatory MFA for all accounts
- Allowed methods:
    * TOTP
    * Hardware key (FIDO2)
    * Secure mobile push
- Not allowed:
    * SMS fallback for high-risk accounts
